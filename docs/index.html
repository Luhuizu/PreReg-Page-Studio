PRE-REGISTRATION STATIC LANDING PAGE GENERATOR

PreReg Page Studio 是一個給營運 / 行銷使用的 內部設定介面。透過表單填寫遊戲資訊、上傳符合 RWD 標準的圖片（桌機 / 平板 / 手機），系統可以輸出統一格式的設定檔與靜態頁面資源，減少前端工程師重複切版工時，提升新遊戲預約頁上線速度與一致性。

OVERVIEW ｜ 專案說明
這個工具在做什麼？
簡單來說，這是一個 「預約頁產生器的設定後台」：
- 左側：營運人員填寫遊戲資訊、選擇風格、上傳圖檔。
- 右側：即時預覽預約頁在手機上的呈現。
- 最後：產生一份結構化設定（JSON / 靜態檔），交給前端或自動流程去輸出正式預約頁。
  • 支援多遊戲、多專案的預約頁產出。
  • 依照公司既有風格，目前規劃兩種 Template： 冒險 / 航海 與 宮廷 / 古風。
  • 每個圖片欄位皆導向 RWD 概念：桌機 / 平板 / 手機 個別上傳，避免僅用單一圖檔硬縮放。
FEATURES ｜ 主要功能
1. 遊戲資訊設定
- 遊戲中 / 英文名稱
- Slogan、一句話介紹
- 詳細介紹（支援多行）
- 類型 / 標籤（方便未來分類或 SEO）
2. 預約檔期設定
- 預約開始 / 結束時間
- 預計上線日
- 是否顯示倒數計時
- 後續可延伸至後台自動關閉預約表單
3. RWD 圖片與 Icon 上傳
- 主視覺背景（桌機 / 平板 / 手機）
- 預約按鈕圖片（多尺寸）
- 遊戲 Logo、平台 icon（iOS / Android）、其他活動 icon
- 以「群組」方式呈現，方便營運理解與管理。
4. 主題與風格切換
- Template A：冒險 / 海洋風（類似 Shiny Voyage）
- Template B：宮廷 / 古風（沿用既有宮廷遊戲視覺）
- 自訂主色 / 副色 / 按鈕色、字型選擇。
No-code 預約頁設定介面 支援多遊戲 / 多專案 符合 RWD 的圖片規格 產出靜態檔與設定檔
VALUE ｜ 對團隊的幫助
為什麼值得做？
從營運與技術兩個角度，可以這樣看這個工具：
  • 營運 / 行銷： 不需要會 HTML / CSS，也能主導預約頁內容與更新。
  • 前端工程： 不用一再重複切相似的 Landing Page，專心維護少數 Template 即可。
  • 管理面： 新遊戲預約頁上線節奏更穩定，可預測、可複用、好交接。
  • 長期： 未來可接 AI 產圖 / 產文流程，變成「AI + 設定」一鍵產出預約頁的完整解決方案。

FLOW ｜ 使用流程（營運視角）
  1. 選擇遊戲專案，填寫基本資訊（名稱、介紹、標籤）。
  2. 設定預約檔期與是否顯示倒數計時。
  3. 依照系統提示，上傳桌機 / 平板 / 手機主視覺與按鈕圖。
  4. 選擇主題（冒險 / 宮廷）與顏色，確認右側預覽畫面。
  5. 按下「產生設定檔」，交由前端 / 自動流程輸出正式預約靜態頁。
