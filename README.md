# PreReg Page Studio

An internal web tool for configuring and generating game pre-registration static pages.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Responsive admin-style UI
- Real-time preview of configuration
- Comprehensive form sections for:
  - Basic game information
  - Pre-registration settings
  - Theme and styling
  - Image assets (with RWD support)
- JSON configuration preview
- Type-safe configuration state
